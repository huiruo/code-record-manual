#### 2.关于问题_行动 - 2022.05.08

最近通过笔试意识到，编程的根基在于写代码，不是：
- 了解了
- 遇到问题能通过百度谷歌解决问题

所以说基础差,原因:
1. 不去实践,满足停留在表面（或则通过搜索解决问题）
<br>把问题记录也来不是真的懂得，一定要亲自去探究，亲自实现(纸终浅，知要躬行）
2. 东西太繁杂，困难太多,新的一天开始继续走老路

解决和计划：行动
```
别想太多，就是写代码，别通过百度谷歌解决问题。
一句话三个要求。
```
