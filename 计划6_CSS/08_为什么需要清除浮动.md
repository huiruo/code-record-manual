

## 清除浮动的本质:主要为了解决父级元素因为子级浮动引起内部高度为0的问题。
```
准确地说，并不是清除浮动，而是清除浮动后造成的影响

再解释下就是在标准流下面一个父p没有设置高度属性,那么它的高度就会被子元素的高度撑开。但是如果这个父p中的子元素是浮动的话，如果父p下面再有一个兄弟p，那么这个兄弟p就会遮挡这个父元素。这个现象也叫浮动溢出。
```

```css
.clearfix:after{ 
   display: block;
   content:'';
   clear: both;
}

.content:after {
  display: table;
  content: "";
  clear: both;
}
```
